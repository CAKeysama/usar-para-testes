Comandos do Git

Clonando

Inicilização das atividades

  • git fetch --all

    Sincronizar repositorios local com repositorios remoto.

  • Git pull

    Faz o dowload das atualizações do servidor remoto.

Finalização das atividades

  • git status

    lista as motidificações

  • Git add .

    Adiciona sa modificações nos servidor local.

  • Git commit -m "(texto)"

    Monta as modificações para envio ao servidor remoto.

  • git push

    Faz upload das modificações do servidor local para o servidor.
